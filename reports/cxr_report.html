This report is a running log of all the progress being made at detecting normals in the Chest X-Ray project.

The Data¶

The data is 400,000 cases of plain film chest x-rays (CXRs). Below is an update of the import dates that have happened:

  • 2017/03/27: CXR pull at rate of ~4000 CXRs
  • 2017/04/06: Pause CXR pull. Total pulled = ~50,000 cases.

Labels:

  • Label 0: Normals (~65% of the data)
  • Label 1: Non-Normals (~35% of the data)
  • Label 2: Emergents (~0.1% of the data)

Initial Results¶

We have created some initial results for the Chest X-Ray project. For all initial results, we did the following:

Pipeline:

  1. Test on randomly chosen 5,000 cases.
  2. Preprocess Data:
    1. Resize image to 512x512
    2. Z-score normalize image
    3. clip z-score at [-3,3]
  3. Run GoogLe Network
    • Dropout: 0.5
    • Learning Rate: 1e-3
    • L2 Regularization: 1e-12
    • LR decay: None
    • Batch Size: 128
  4. Run Training on 1k, 3k, 10k, and 50k datasets to see if dataset size matters

import dicom
import numpy as np
from skimage import exposure

import matplotlib.pyplot as plt
%matplotlib inline

# Filepath
path_dcm = 'figures\\examples\\FO-8313167968660364988.dcm'

# Reading in the dicom image.
dcm = dicom.read_file(path_dcm)
dcm = dcm.pixel_array
dcm = dcm.astype(np.float32)
dcm -= np.min(dcm)
dcm /= np.max(dcm)
dcm = 1 - dcm

# Doing histogram normalization.
img = exposure.equalize_hist(dcm)

# Displaying Everything
plt.rcParams['figure.figsize'] = (12,8)
f,ax = plt.subplots(1,2)
ax[0].imshow(dcm)
ax[0].axis('off')
ax[0].set_title('Original')
ax[1].imshow(img)
ax[1].axis('off')
ax[1].set_title('High Contrast')
